1. 一般不在child中使用relative改变top, left...因为这样会使它脱离原本的位置流变成游离的模块并有可能覆盖其他模块

2. 当你想要子模块中的某个child相对于这个parent的位置固定，那么最简单的方法就是在parent中添加 position： relative，
此时，子模块中的positon: absolute 将会是相对于parent的绝对位置而不是相对于整个页面的绝对位置。

3. 因此，例如如果我们想要让header固定在顶部，并保持内部的所有标题和nav bar都位置固定就可以在header中设定 position： relative

4. 并不是只有relative能做到这一点， 除了static 都可以达到这样的效果

5. fixed: 始终固定位置（即使滑动页面）

6. sticky: 初始页面-static ， 滑动后-fixed
